# 👨🏻‍💻Webserv🧑🏼‍💻

This project is here to make HTTP server like NGINX.

We follow the HTTP RFC.

## 🛠Installation

`make` on root.

## ▶️Usage

Excute `webserv` binary file with a configure file in the first argument

`./webserv examle.conf`
